# Команды мода "ModStrany" и инструкция по упаковке

Ниже перечислены команды, которые реализованы в моде, и краткое описание их функций.

- `/country create <id> <name>`: Создать новую страну с идентификатором `<id>` и отображаемым именем `<name>`. Даёт стартовый баланс 5 000 000. Требует прав оператора (permission level 2).

- `/country join <id>`: Вступить игроку в страну с идентификатором `<id>`. Выполняется игроком.

- `/country ally create <id> <displayName> <first> <second>`: Создать именованный альянс с уникальным `<id>` и названием `<displayName>`, сразу добавив в него страны `<first>` и `<second>`. Требует прав оператора.

- `/country ally add <allianceId> <country>`: Добавить страну в существующий альянс. Альянс ограничен максимумом 10 стран; операция вернёт ошибку при переполнении или неверных ID. Требует прав оператора.

- `/country ally rename <allianceId> <newName>`: Переименовать альянс (отображаемое имя). Требует прав оператора.

- `/country war <attacker> <defender>`: Объявить войну между странами. Требует прав оператора. Мод запрещает объявлять войну против страны, в которой нет реальных (онлайн) игроков — в этом случае команда отклоняется.

- `/country transfer <from> <to> <amount>`: Перевести деньги между двумя странами. Требует прав оператора. При переводе выполняется списание с одной страны и зачисление другой; при неудаче операция отклоняется.

- `/country money add <amount>`: Добавить указанную сумму в баланс страны, к которой принадлежит выполняющий игрок. Выполняется игроком.

- `/country money remove <amount>`: Списать указанную сумму со страны, к которой принадлежит выполняющий игрок. Выполняется игроком. Траты транслируются в чат сервера.

- `/country info <id>`: Показать информацию о стране (имя, цвет, баланс, количество членов, союзники/враги и т.д.).

- `/country rename <id> <newName>`: Переименовать страну с идентификатором `<id>` в новое отображаемое имя `<newName>`. Требует прав оператора.

Поведение уведомлений:
- Создание/переименование альянсов, объявления войны, переводы и расходы транслируются в общий чат сервера (`broadcast`).

Файловая структура ресурсов (из репозитория):
- Спрайты и текстуры находятся в `s/assets/modstrany/textures/...` и `s/assets/modstrany/models/...`.

Инструкция по упаковке мода (сборка JAR)

1) Требования на машине разработчика:
   - Java 17
   - Gradle (или `./gradlew` в проекте)
   - Копия Minecraft Forge MDK, соответствующая версии `1.20.1-45.1.0` (проект уже настроен на эту версию в `build.gradle`).

2) Рекомендуемый способ (через Gradle wrapper, если он есть):
   - В корне проекта выполнить:

```powershell
./gradlew build
```

   - Результат: файл `build/libs/<project>-<version>.jar`.

3) Альтернативно, используя системный Gradle:
```powershell
gradle build
```

4) Установка в Minecraft локально:
   - Скопируйте собранный JAR в папку `mods` вашего профиля Forge на клиенте или сервере.
   - Запустите сервер/клиент с Java 17 и Forge 1.20.1.

5) Замечания и ограничения в текущем репозитории:
   - В этой среде сборка не выполнялась (Gradle может отсутствовать на хосте). Если хотите, я могу добавить Gradle Wrapper в репозиторий и попытаться собрать мод здесь — подтвердите, и я продолжу.
   - Данные стран и альянсов хранятся сейчас в памяти (не персистятся в сохранениях). Для сохранения между запусками требуется реализация `SavedData` (Forge) — могу добавить по запросу.
